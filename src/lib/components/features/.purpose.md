# **Features**

- **Definition**: Features are **functional units of your app** â€” something that provides a distinct piece of functionality or behavior.
- **Examples**:
  - Authentication (`login`, `signup`, `logout`)
  - User profile management
  - Shopping cart
  - Search bar logic

- **Composition**: A feature may contain multiple UI components, hooks/stores, and sometimes its own API calls.
- **Where to put them**:
  Typically under something like:

  ```
  src/lib/components/features
    auth/
      Login.svelte
      Signup.svelte
      useAuthStore.ts
    cart/
      Cart.svelte
      CartItem.svelte
      useCartStore.ts
  ```

  This keeps your app organized around **capabilities**.
